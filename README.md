# ゲームタイトル
「ゴース撮る」
## 概要
プレイ人数　１人
対応機種　　PC
ターゲット　写真を撮るのが好きな人
制作期間　　3ヶ月

このゲームは森を歩き回り、おばけの写真を撮影するゲームです。
おばけはこちらに気がつくと逃げてしまうので注意を払いながら撮影しなければいけません。
制限時間内に写真を10枚撮って高得点を目指しましょう。

## 使用技術
- C++
- Visual Studio 2022
- DxLib
